# Kit → Flutter — conversion queue

Per-screen prompts generated by `tool/parity/gen_convert_prompts.mjs` from
`tool/parity/convert-prompt.template.md`. Each `NN-<screen>.md` is self-contained.

## How to run (loop)

Dynamic-mode loop, one screen per iteration:

> `/loop` Đọc `docs/agent/kit-to-flutter/README.md`, chọn màn **pending** đầu tiên (ô `[ ]`),
> đọc + thực thi ĐẦY ĐỦ file prompt của màn đó (baseline → verify --full → commit → push),
> rồi đổi ô đó thành `[x]`. Mỗi vòng đúng 1 màn. Nếu prompt bảo DỪNG (drift/divergence) →
> dừng, báo, chờ người. Hết pending → báo hoàn tất.

`dashboard` + `review` đã có slots/states + parity test (2 màn mẫu) → đánh dấu sẵn `[x]`.
